Monocle in C++
#+BEGIN_SRC bash
sudo apt-get install libsdl2-dev libsdl2-image-dev libsdl2-ttf-dev libsdl2-mixer-dev
#+END_SRC

to build:
#+BEGIN_SRC bash
mkdir build
make
#+END_SRC

to install:
#+BEGIN_SRC bash
sudo make install
#+END_SRC

to uninstall:
#+BEGIN_SRC bash
sudo make uninstall
#+END_SRC

A minimal example is provided in [[https://github.com/Accacio/morpheus][morpheus]]
